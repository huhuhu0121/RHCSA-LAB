# RHCSA Lab

RHCSA 자격증 준비 과정에서 진행한 Linux 시스템 관리 실습 내용을 기록한 저장소입니다.  
단순 이론이 아닌 실제 서버 환경을 기반으로 한 실습과 트러블슈팅 과정을 중심으로 정리하고 있습니다.

---

## 🛠 Tech Stack
- Linux (Rocky Linux / Ubuntu)
- Virtualization (VirtualBox)
- SSH
- Bash

---

## 📚 Study Contents

### 1. System Management
- 사용자 및 그룹 관리
- 권한 및 소유권 설정
- 프로세스 및 서비스 관리 (systemctl)

### 2. Storage
- 디스크 관리
- LVM 구성 및 확장

### 3. Network
- IP 설정
- 포트 및 방화벽 관리

### 4. Service
- SSH 설정
- 웹 서버(Nginx) 기초

---

## 🔧 Troubleshooting

실습 중 발생한 문제와 해결 과정을 기록합니다.

- 서비스 실행 오류 해결
- 포트 충돌 문제 해결
- 권한 설정 오류 해결
